Medical Face AI Analyzer

A Flutter-based tool built for aesthetic clinics to assist doctors in analyzing facial areas and planning cosmetic procedures.

  • Patients upload a face image for analysis.
  • MediaPipe Face Mesh detects 468 facial landmarks in real-time, enabling accurate region mapping.
  • Doctors can select default facial areas (e.g., lips, jaw, cheeks) or freely drag to define custom zones.
  • Each selected region is outlined using triangle meshes from the landmark data.
  • The app supports interactive highlights and tap gestures for zone-based annotations.
  • Built using Flutter with custom drawing logic and gesture handling.
