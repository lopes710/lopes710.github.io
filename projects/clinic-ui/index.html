Medical Aesthetic Clinic UI

This Flutter UI was designed for a medical aesthetic clinic, allowing users to track key lifestyle inputs like sleep, hydration, and activity. All graphs, animations, and calendar components were custom built programmatically.

  • Custom animated graphs for 7-day progress tracking
  • Fully interactive calendar UI, built from scratch
  • Modular and visually appealing layout for mobile
  • Programmatic animation of transitions and charts
  • Designed to collect data for aesthetic treatment planning
