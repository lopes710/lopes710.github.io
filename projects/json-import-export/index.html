JSON Import/Export Editor

This Flutter-based admin tool allows seamless syncing of structured JSON content with a backend database. The interface supports editing treatment data, sending updates to a Flask backend, and validating changes via JSON export.

  • Connected to a Flask API with backend validation
  • Supports live editing of treatment data
  • Exports updated content as downloadable JSON
  • Useful for internal tooling and content migration
  • Built entirely in Flutter with fullstack integration
