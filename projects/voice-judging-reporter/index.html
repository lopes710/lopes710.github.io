Voice Judging Reporter

Flutter app for dog show judges to record their voice evaluation and automatically generate a structured report.

  • Built with Flutter and Dart
  • Tap-to-record voice input using built-in mic
  • Speech transcribed using Gemini AI
  • Transcription converted into a structured judging report
  • Auto-scores areas like hair, behavior, and posture
  • Report preview with option to review or edit before finalization
  • Ideal for fast, hands-free reporting in competitive settings
